❓ Ayuda

🚀 Cómo usar ShortsGen Pro

  1. Escribe tu guion en el paso 1. Puedes usar una plantilla.
  2. Configura tu voz ElevenLabs en el paso 2. Necesitas una API Key gratuita.
  3. Personaliza el diseño con gradientes, filtros y subtítulos.
  4. Genera y descarga tu Short en formato MP4.

🔑 API Key de ElevenLabs

  • Ve a elevenlabs.io y crea una cuenta gratuita.
  • En Settings → API Keys genera una nueva key.
  • Pega la key en el paso 2 de la app.
  • La versión gratuita incluye 10,000 caracteres/mes.

⚠️ Notas importantes

  • La generación de video usa el procesador de tu dispositivo.
  • En móviles, usa duraciones cortas (15-30s) para mejor rendimiento.
  • Opera y Chrome funcionan mejor que Safari para esta app.
  • Los videos se guardan localmente en tu navegador.
